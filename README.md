# Palash S. Sarate - Personal Website

This is a premium, high-fidelity personal website built with **Next.js** and **Vanilla CSS**, optimized for hosting on **GitHub Pages**.

## Features
- **Next.js App Router**: Modern foundation.
- **Static Export**: Lightning fast performance.
- **Premium Design**: Dark theme with glassmorphism and smooth animations.
- **Responsive**: Mobile-first design.

## How to Deploy to GitHub Pages

1. **Build the site**:
   ```bash
   npm run build
   ```
   This will generate a `out` directory.

2. **Deploying**:
   - You can use the `gh-pages` package or manually push the `out` directory to a `gh-pages` branch.
   - Alternatively, configure GitHub Actions to build and deploy automatically (recommended).

## Local Development
```bash
npm run dev
```
